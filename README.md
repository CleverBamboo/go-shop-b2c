## Go语言 B2C 商城
技术栈：GO、Beego、silenceper/wechat

简单介绍：一个用Go语言Beego框架写的API服务端，用Vue写前端的B2C商城

## 目录结构
* common
  公共代码函数  
  
* conf 
  配置文件

* controllers
  Controller层
  
* helpers
  工具函数
  
* model_views
  展示给前端的Struct
  
* routers
  路由
  
* setting
  后台xml配置，go-shop-b2c.xml
